FAQ

Common Questions

How fast will my site load?
We target a 90+ Lighthouse score on every build - typically < 2s First Contentful Paint on 4G. Images are responsive, JS is code-split, and we run a perf check before handoff.
Do you use WordPress or custom?
Depends on you. WordPress / Strapi for content-heavy marketing sites you want to edit yourself. Next.js / React for web apps and storefronts where you need control and speed.
Can you redesign my existing site?
Yes - we can rebuild a tired site without losing SEO. We map URLs, set up 301 redirects, preserve content structure and improve Core Web Vitals.
Is hosting included?
Setup is - we deploy to Vercel / Netlify / your AWS / your CMS. You own the accounts. Ongoing hosting is paid directly to the provider, usually $0-20/month for a marketing site.
Do I get analytics?
Yes - Google Analytics, Plausible or PostHog set up before launch, with the key events you care about already tracked.
